# Travel Stream

A web app to streamline travel planning, available at [travelstreamapp.com](https://travelstreamapp.com).

## Development

### Requirements

- Node.js version 22

### Installation

- Install dependencies using `npm install`

### Set up

- Start the Vite development server using `npm run dev`

### Tests

- Run tests using `npm test`

## Deployment

The app is deployed with Cloudflare Pages.

### Set up

- None

### Process

- A GitHub Action will deploy new commits on `main` to Cloudflare Pages automatically.
